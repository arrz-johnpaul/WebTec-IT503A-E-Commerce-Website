# Shoe E-Commerce Website

This project is a front-end prototype of an e-commerce website focused on selling shoes. It was created as part of our Web Technology (WebTec) course to apply web development concepts using modern front-end technologies.

---

## Project Overview

The website includes several core pages that simulate the basic user experience of an online shoe store:

* **Landing Page** – Serves as the homepage, showcasing featured products and promotions.
* **Product List Page** – Displays all available shoes with their details such as names, images, and prices.
* **Product Details Page** – Provides more information about a selected product.
* **Shopping Cart Page** – Allows users to view and modify items before proceeding to checkout.
* **Checkout Page** – Simulates the order finalization process.

---

## Technologies Used

* **HTML5** – Structure and semantic layout of pages
* **CSS** – Styling and responsive design
* **Bootstrap 5** – Framework for faster and cleaner front-end development
* **JavaScript (ES6)** – Adds interactivity and dynamic content updates
* **JSON** – Used for managing and loading product data

---

## Features

* Responsive and mobile-friendly design
* Interactive shopping cart and checkout flow using JavaScript
* Organized product data handled through JSON
* Simple, clean, and modern layout using Bootstrap 5 components

---

## Project Scope

This is a **front-end only project**. It focuses on layout, design, and user interaction without backend or database functionality. It serves as a prototype to demonstrate e-commerce page structure and functionality using web technologies.
